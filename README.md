AI-powered search for every Indian, in every language.

## What it does
Ask questions in Hindi, Kannada, Tamil, or English 
and get direct AI-powered answers instantly.

## Why it matters
500 million Indians speak regional languages.
Most AI tools only work well in English.
BharatSearch bridges that gap.

## Built with
- Google Gemini API
- Gradio
- Python

## How to run locally
1. Clone this repo
2. Install requirements: pip install -r requirements.txt
3. Add your Gemini API key
4. Run: python app.py
